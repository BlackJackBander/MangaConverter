# MangaConverter

Это веб-приложение предназначено для преобразования изображений (JPG, PNG) и ZIP-архивов с изображениями в PDF-документы. Основная цель - удобная конвертация комиксов, манги и других коллекций изображений в единый PDF-файл.

## Ключевые возможности

### 1. **Двойной режим загрузки**
- **Режим изображений** - загрузка отдельных файлов JPG/PNG
- **Режим ZIP-архивов** - загрузка ZIP-файлов с изображениями внутри

### 2. **Управление изображениями**
- Просмотр миниатюр загруженных изображений
- Предпросмотр выбранного изображения
- Управление порядком изображений (перемещение вверх/вниз)
- Удаление отдельных изображений
- Отображение информации о файлах (название, размер)

### 3. **Интеллектуальная конвертация в PDF**
- Автоматическое масштабирование изображений под размер страницы PDF
- Сохранение пропорций изображений
- Центрирование на странице
- Поддержка многостраничных PDF-документов

### 4. **Пользовательский интерфейс**
- Современный адаптивный дизайн с градиентами
- Drag-and-drop загрузка файлов
- Визуальная обратная связь (статусы, загрузка)
- Поддержка мобильных устройств

## Технические особенности

### Архитектура
- Чистый JavaScript (ES6+) с использованием классов
- Библиотеки: jsPDF для создания PDF, JSZip для работы с архивами
- Модульная структура с классом `PDFConverter`

### Ключевые функции класса PDFConverter

1. **`processZipFile()`** - извлечение изображений из ZIP-архивов
2. **`convertToPDF()`** - основная логика создания PDF-документа
3. **`addImage()`** - добавление и обработка изображений
4. **`showImage()`** - управление предпросмотром
5. **`moveImage()`** - изменение порядка изображений

### Особенности реализации
- Оптимизированная обработка больших объемов изображений
- Корректное освобождение URL объектов после удаления
- Обработка ошибок при загрузке и конвертации
- Автогенерация имени PDF-файла на основе исходного ZIP

## Преимущества приложения
- **Удобство** - не требует установки дополнительного ПО
- **Безопасность** - вся обработка происходит локально в браузере
- **Производительность** - эффективная работа с большими наборами изображений
- **Кроссплатформенность** - работает на любом устройстве с современным браузером

Приложение идеально подходит для фанатов комиксов и манги, позволяя легко организовывать коллекции изображений в удобные PDF-документы для чтения на различных устройствах.
